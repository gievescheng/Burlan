# 柏連稽核系統位置說明

這一份才是之後要繼續使用的柏連專用系統資料夾：

`C:\Users\USER\Documents\Codex\ISO 9001 SOP-柏連正式版\_柏連稽核系統`

說明：

- 柏連正式文件仍然放在上一層資料夾
- 系統程式獨立放在 `_柏連稽核系統`
- 之後請不要再到 `C:\Users\USER\Documents\Codex\自動稽核程式` 修改柏連內容

常用檔案：

- 啟動檔：`C:\Users\USER\Documents\Codex\ISO 9001 SOP-柏連正式版\_柏連稽核系統\啟動柏連稽核系統.bat`
- 畫面檔：`C:\Users\USER\Documents\Codex\ISO 9001 SOP-柏連正式版\_柏連稽核系統\index.html`
- PDF 解析核心：`C:\Users\USER\Documents\Codex\ISO 9001 SOP-柏連正式版\_柏連稽核系統\v2_backend\app\pdf_extractors.py`

目前設定：

- 系統程式資料夾：`_柏連稽核系統`
- 文件根目錄：上一層的 `ISO 9001 SOP-柏連正式版`

也就是說，系統會優先把柏連正式版資料夾當成文件來源，而不是潔沛那個專案資料夾。
